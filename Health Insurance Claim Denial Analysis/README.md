**Predicting Health Insurance Claim Denials**
A project completed in one of my courses to learn more about optimizing models and assessing their performance.

**Introduction**

This project aims to see if medical insurance claim denials can be predicted based on various factors about a medical visit including amount billed, insurance and type of submission. The data came from a Kaggle data set that uses data that simulates real-world scenarios.

**Background**

Denied health insurance claims have a negative effect on both patients and medical providers/hospitals.  They can prevent patients from getting their medications on time and delay reimbursements to hospitals or patients. This project aims to see if there is a way to determine if an insurance claim may be denied based on other information about the claim including factors like location or insurance type in order to prevent insurance claim denials from happening.

**Data**

Link to Kaggle data set: https://www.kaggle.com/datasets/leandrenash/enhanced-health-insurance-claims-dataset
