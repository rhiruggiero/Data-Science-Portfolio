**Crohn's Disease and Ulcerative Colitis Flare Prediction**

A project completed during my final course in my master’s in data science degree.

**Introduction**
This project uses a Kaggle dataset referenced below to see if there is a correlation between various gut microbiome levels and flares in Crohn’s Disease and Ulcerative Colitis patients. The Kaggle data is based on a study, also referenced below, completed on 132 patients that tested their various gut bacteria levels once a week for an entire year to see how they changed.

**Background**
A patient is in a flare when they have active symptoms of inflammation, and they can be represented by an increase in an inflammatory marker called fecal calprotectin. Increased inflammation can cause permanent damage, and getting treatment quickly once a flare begins is important to ensure that the disease doesn’t progress.  This project aims to see if there are any gut bacteria that could potentially correlate with a person about to go into a flare.  By knowing how bacteria levels change, medical professionals would be able to intervene earlier and potentially prevent the flare altogether.

**Data**
Link to Kaggle dataset:  https://www.kaggle.com/datasets/qasimhu/human-gut-microbiome-atlas-hmp2
Link to the study the data came from: https://ibdmdb.org/

This project uses random forest modeling to see which bacteria may be correlated with flares.
